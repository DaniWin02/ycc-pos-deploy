# YCC Admin Panel

## 📋 Overview

El Admin Panel es una aplicación web moderna para la gestión administrativa completa del sistema YCC POS. Proporciona una interfaz intuitiva para administrar productos, inventario, ventas, usuarios y reportes del restaurante.

## � Features

### Core Features
- **Dashboard en Tiempo Real**: Métricas clave con actualizaciones automáticas
- **Gestión de Menú**: CRUD completo de productos, categorías y modificadores
- **Control de Inventario**: Seguimiento de stock y alertas de bajo inventario
- **Análisis de Ventas**: Reportes detallados y gráficas interactivas
- **Gestión de Usuarios**: Control de roles y permisos
- **Configuración de Tiendas**: Multi-sucursal con switching en tiempo real
- **Alertas en Vivo**: Notificaciones automáticas del sistema

### UX Features
- **Sidebar Colapsable**: Navegación eficiente con iconos intuitivos
- **Tema Profesional**: Diseño claro con colores corporativos
- **Responsive Design**: Optimizado para desktop y tablets
- **Animaciones Suaves**: <300ms, 60fps performance
- **Accessibility**: WCAG 2.1 AA compliance

### Technical Features
- **TypeScript**: Tipado estricto para mejor mantenimiento
- **TanStack Query**: Data fetching optimizado con cache
- **Framer Motion**: Animaciones performantes
- **Modular Architecture**: Componentes reutilizables
- **Real-time Updates**: WebSocket integration
